# Pakistan Sales Performance Dashboard - FY2026

## 📊 Project Description

A comprehensive **Excel-based sales analytics dashboard** for Pakistan operations in FY2026.
This dashboard provides real-time insights into revenue, profit, product performance, and sales team metrics.

**Business Value:**
- Track total revenue of **$102K** across 4 regions
- Monitor profit margins at **40%** efficiency
- Analyze performance of **240 total orders**
- Evaluate **6 sales representatives** performance

## 📖 How to Use This Dashboard

### **Step 1: Open the Dashboard**
1. Download the PDF file
2. Open in Adobe Reader or any PDF viewer
3. Zoom in to see detailed charts and metrics

### **Step 2: Navigate Dashboard Pages**

**Page 1 - Executive Summary:**
- View key performance indicators
- Analyze monthly profit vs revenue trend
- Check sales rep leaderboard
- Review product performance

**Page 2 - Detailed Analysis:**
- Examine category heatmap
- Study discount vs margin relationship
- Analyze customer segment breakdown
- Review payment method usage
- Check city-wise order distribution

### **Step 3: Use Interactive Filters**

The dashboard includes filters for:
- 📍 **Region** (East, North, South, West)
- 📦 **Category** (Accessories, Furniture, Office Supplies, Technology)
- 🛒 **Sales Channel** (Distributor, Online, Retail Store)
- 👥 **Customer Segment** (Consumer, Corporate, Small Business)
- 👤 **Sales Rep** (Ayesha, Bilal, Fatima, Hamza, Mariam, Omar)
- 💳 **Payment Method** (Bank Transfer, Cash, Credit Card, Mobile Wallet)
- 📅 **Order Date** (January - December 2025)

### **Step 4: Key Insights to Look For**

✅ **Best Performing Region:** North ($11K revenue)
